# Title

### Properties

| Property | Description    | Type                                   | Default |
| -------- | -------------- | -------------------------------------- | ------- |
| level    | header level   | `1` \| `2` \| `3` \| `4` \| `5` \| `6` | `3`     |
| align    | text alignment | `left` \| `center` \| `right`          | `left`  |

### Basic usage
