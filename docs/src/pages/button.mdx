## Button

### Properties

| Property | Description                    | Type                               | Default  |
| -------- | ------------------------------ | ---------------------------------- | -------- |
| label    | Button Text                    | `string`                           |          |
| status   | The color of the button        | `default` \| `primary` \| `danger` |          |
| type     | What type of button to display | `default` \| `link` \| `outline`   |          |
| size     |                                | `small` \| `normal` \| `large`     | `normal` |
| loading  | show loading status            | `boolean`                          | `false`  |

## Loading

      <div>
